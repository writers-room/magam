⚙️ 대시보드 설정
채팅 · 테마 · 타이머 · 개인정보 설정을 한 곳에서 정돈해요.
채팅 가로 넓이
현재: px
레이아웃
작업 흐름에 맞춰 "메인 + 채팅"의 위치를 바꿀 수 있어요.
테마 선택
동그라미는 "배경(왼쪽) + 내 말풍선(오른쪽)" 반반 미리보기예요.
타이머 표시
타이머는 잔잔한 그라데이션 + "임박" 때만 살짝 강조돼요.
임박 강조 기준
마감 분 이하면 강조가 들어가요.
내 데이터
개인 작업 영역(투두/메모/목표/오늘 누적)은 Firebase에 필명 기준으로 저장돼요.
필명이 같으면 덮어써질 수 있으니, 중복되지 않는 필명을 권장해요.
로컬 설정 초기화
테마/채팅 너비/레이아웃/타이머 표시 설정만 초기화돼요.
